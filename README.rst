星际争霸2 裁判合约
===================


:Date: 01/06 2017

.. contents::

介绍
-----

构建在 `Ethereum BlockChain` 上面的裁判 智能合约。

该合约可以对 `星际争霸2` 这款游戏的比赛结果进行分析，并给出胜负结果。
